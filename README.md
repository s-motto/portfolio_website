# 🦦 Sabrina Motto — Portfolio

My personal portfolio site. Nothing groundbreaking, but it's mine.

🔗 **Live site**: [sabrina-motto.vercel.app](https://sabrina-motto.vercel.app/)

---

## About

A single-page portfolio built from scratch to showcase my projects and skills. Designed and developed independently as part of my ongoing self-study in web development.

---

## Built With

- HTML5
- CSS3 (custom, no frameworks except where noted)
- Vanilla JavaScript
- Bootstrap 5 (for the project cards grid)
- Google Fonts — [Sono](https://fonts.google.com/specimen/Sono)
- [Devicon](https://devicon.dev/) for tech stack icons
- [Font Awesome](https://fontawesome.com/) for UI icons

---

## Features

- Responsive layout with mobile hamburger menu
- Diagonal section design with alternating clip-path layout
- Project cards with modal popups (description, GitHub link, demo link where available)
- Skills & Tools section with technology icons
- Contact section with links to email, GitHub and LinkedIn

---

## Project Status

Ongoing — new projects will be added as they are completed. The site itself is also one of the listed projects, which feels a bit recursive but here we are.
